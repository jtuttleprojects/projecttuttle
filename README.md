# Cybersecurity & Networking Project Portfolio

## Overview

This repository contains a collection of hands-on cybersecurity, networking, and IT infrastructure projects. These projects demonstrate practical experience in network design, security analysis, and system troubleshooting using industry-relevant tools and environments.

The goal of this portfolio is to showcase real-world technical skills applicable to roles in Network Operations, Cybersecurity, and Operational Technology (OT).

---

## Projects

### 1. Network Infrastructure Lab (Packet Tracer)

**Description:**
Designed and implemented a multi-router network using Cisco Packet Tracer to simulate an enterprise environment.

**Key Skills:**

* Routing (RIP v2)
* DHCP & DNS configuration
* Subnetting and IP planning
* Network troubleshooting

**Highlights:**

* Built 3 interconnected subnets with full connectivity
* Configured dynamic routing between routers
* Implemented centralized DHCP services
